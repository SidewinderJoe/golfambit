Steps for Script
--User enters miles and zipcode

1) Create Variable to Convert Miles to Kilometers (API does not support miles)
2) Create Variable to Lookup longitude and latitude of zipcode
3) Put Variables into api request and get JSON file
4) Take JSON and populate page and save list to sql database
